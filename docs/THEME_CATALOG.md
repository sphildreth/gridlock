# Theme Catalog

Decent Bench now ships a bundled starter library of twenty complete TOML themes.
The collection balances dense workbench defaults, familiar modern editor styles,
crisp light options, and accessibility-focused high-contrast themes.

## Recommended defaults

- `classic-dark`: safest first-run default for dense desktop database workflows.
- `nordic-night`: approachable modern dark showcase with soft contrast.
- `frost-light`: crisp modern light option for users coming from editor-style UIs.
- `high-contrast-dark`: strongest accessibility-oriented dark theme.

## Collection balance

- 10 dark themes, spanning classic workbench, modern editor, and expressive families.
- 6 light themes, mixing dense enterprise, paper-like, cool, and neutral styles.
- 4 high-contrast themes focused on focus rings, separators, and grid clarity.
